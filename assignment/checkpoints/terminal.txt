# 1 - What is the command for listing the files in the current folder
(directory)?
ls

# 2 - What is the command for changing directories?
cd

# 3 - What is the command for going "up" a folder (to a parent directory)?
cd ..

# 4 - What is a quick way to create a new folder?
mkdir

# 5 - What output would you expect when you run the command 'ls' in an empty
folder? What about a folder containing files?
In an empty folder, a new line will appear that has nothing in it. In a folder
that contains
files it will list the files.

# 6 - What is a quick way to create a new file?
the 'touch' command

# 7 - Run 'git --version' in your terminal. What is the output?
git version 2.30.1 (Apple Git-130)

# 8 - Why would you run the command 'git status'?
To show if and what files have been modified and if changes are ready to
be committed

# 9 - What does the command 'git add .' do?
This command adds everything in the working directory to git as tracked and
stages changes to be committed

# 10 - What is the command to push your committed changes to your repo on GitHub?
git push
